## 💡 Kesimpulan Perjalanan Belajar (Pertemuan 1-7)

Kesimpulan Umum:
- **Pertemuan 1: Pengenalan Data Science**: DData Science merupakan sinergi antara matematika/statistika, pemrograman, dan pemahaman bisnis untuk mengubah data mentah menjadi keputusan strategis. Prosesnya dijalankan secara iteratif lewat metodologi CRISP-DM—mulai dari membedah kebutuhan bisnis hingga penerapan model di dunia nyata. Ekosistem kerjanya mengandalkan Python dan Google Colab untuk eksperimen kode, serta GitHub sebagai ruang kolaborasi dan pameran portofolio.
- **Pertemuan 2: Struktur Data Python, NumPy & Pandas**: Efisiensi pengolahan data bertumpu pada penguasaan tools yang tepat. Python menyediakan struktur data bawaan seperti List, Tuple, Dictionary, dan Set untuk manajemen objek dasar. Di atasnya, NumPy hadir mempercepat komputasi numerik berdimensi tinggi lewat operasi vektor, sementara Pandas DataFrame mempermudah manipulasi data tabular seperti memfilter, mengelompokkan, dan merangkum data secara intuitif.
- **Pertemuan 3: Data Cleaning**: Prinsip utama pemodelan adalah menjaga validitas input karena kualitas performa model ditentukan oleh kebersihan datanya. Proses ini berfokus pada pembersihan data dari missing values, penghapusan data ganda, serta penjinakan pencilan (outliers) menggunakan metode IQR atau Z-Score. Di samping itu, dipelajari juga cara memperluas cakupan data dengan menarik dataset eksternal dari format JSON dan REST API menggunakan pustaka requests.
- **Pertemuan 4: Statistika Dasar & Analisis Data**: Statistika deskriptif adalah lensa untuk memahami pola tersembunyi pada data. Melalui metrik pemusatan (seperti mean dan median) serta penyebaran (seperti standard deviation), kita dapat memetakan bentuk distribusi data berdasarkan nilai skewness dan kurtosis. Analisis ini diperdalam secara visual: univariat untuk melihat sifat tunggal variabel (via boxplot/histogram), dan bivariat untuk mengukur keeratan hubungan antar-variabel menggunakan koefisien korelasi.
- **Pertemuan 5: Visualisasi Data**: Visualisasi bukan sekadar menghias grafik, melainkan menyampaikan cerita yang jujur, estetis, dan mudah dipahami. Memanfaatkan fleksibilitas Matplotlib dan keindahan siap pakai dari Seaborn, grafik dirancang untuk menyajikan konteks yang jelas. Setiap temuan visual kemudian dibedah menggunakan kerangka berpikir "What? So what? Now what?" guna mengubah grafik abstrak menjadi rekomendasi konkret bagi pemangku kepentingan.
- **Pertemuan 6: Persiapan Data**: SSebelum algoritma dijalankan, data harus melewati tahap standardisasi agar siap dikonsumsi oleh model machine learning. Tahap ini meliputi konversi data teks menjadi numerik (encoding) dan penyetaraan skala angka (scaling) agar tidak ada fitur yang mendominasi secara tidak adil. Aturan krusial di sini adalah wajib membagi data (train-test split) di awal sebelum melakukan pengskalan untuk menghindari kebocoran data yang bisa merusak objektivitas evaluasi.
- **Pertemuan 7: Pengantar Regresi Linear & Evaluasi Model**:  Sebagai langkah awal dalam predictive analytics, Regresi Linear digunakan untuk memproyeksikan nilai kontinu masa depan berdasarkan tren variabel saat ini. Validitas prediksi tersebut kemudian diuji menggunakan metrik evaluasi seperti MAE, RMSE, dan $R^2$. Proses ini memastikan bahwa model yang dibangun tidak hanya bekerja di atas kertas, tetapi juga memiliki tingkat akurasi dan toleransi eror yang aman saat diimplementasikan pada kasus nyata.
